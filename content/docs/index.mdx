# Flutcn - A Modern Flutter UI Toolkit

Flutcn is a flexible and customizable UI component library for Flutter, inspired by shadcn/ui. It allows developers to easily integrate high-quality widgets into their applications without bloating their projects.

## Why Flutcn?
- **Modular Components** - Install only what you need.
- **Customizable** - Easily style components to fit your design system.
- **Lightweight** - No unnecessary dependencies.
- **Easy to Use** - Simple API for seamless integration.
